The Projects

🎧 Vibe Machine

Real-time audio visualizer

A real-time, audio-reactive music visualizer built entirely with vanilla JavaScript, HTML Canvas, and the Web Audio API. No frameworks. No build tools. No npm install. Five visualization modes — Bars, Waveform, Circular, Particles, and Starfield — with drag-and-drop audio, keyboard shortcuts, and fullscreen Vibe Mode. Also ships as a desktop Electron app.

JavaScript Canvas 2D Web Audio API Electron Node.js

🎬 Co-Stars

Actor→movie connection game

A full-stack actor connection game — find the shortest path between any two actors through the movies they share. React + TypeScript frontend with offline snapshot caching, powered by a FastAPI backend with type-agnostic pathfinding over a TMDB-enriched SQLite graph. Features levels, hints, suggestions, and custom level creation.

React TypeScript FastAPI SQLite Vite

🤖 Multi-Agent Lab

Document analysis pipeline

Paste in any text, watch four agents analyze it in real time. Extractor, Summarizer, Critic, and Sentiment agents process documents sequentially through a shared pipeline, streaming results to an interactive dashboard via SSE. 100% local, pure Python — no cloud APIs, no LLMs. Fast (~50ms total pipeline).

Python FastAPI SSE Dashboard

🏺 APK Archeologist

Mobile game preservation CLI

A CLI tool for mobile game preservation. Feed it an APK and it produces decompiled source, a catalog of every server endpoint, authentication flow analysis, extractable asset inventory, and a runnable mock server scaffold. Built so games can keep running after their original servers go dark. Supports Unity, Java/Kotlin, with Unreal and Godot planned.

TypeScript Node.js CLI Reverse Engineering
