Vibe Machine

Real-time audio visualizer — zero dependencies, pure vibes.

Try it out on the home page

What Is This

A browser-based audio visualizer built entirely with vanilla JavaScript, HTML Canvas, and the Web Audio API. No frameworks. No build tools. No npm install. Drop in any audio file and watch it come alive across five visualization modes.

Vanilla JS Canvas 2D Web Audio API Node.js Electron

Features

🎨 5+ Visualization Modes

Bars, Waveform, Circular, Particles, Starfield, Starry Night, Pixel Grid, Sunset, and Piano — each a standalone rendering module driven by real-time frequency data.

🎵 Drag & Drop Audio

Drop any audio file onto the canvas. The Web Audio API's AnalyserNode extracts frequency data at 60fps and feeds it directly into the active visualizer's draw loop.

⌨️ Keyboard Shortcuts

Switch modes, toggle fullscreen Vibe Mode, skip tracks, adjust volume — all without touching the mouse. Built for flow state.

🖥️ Electron Desktop App

Ships as both a web app and an Electron desktop application. Same codebase, same visualizers, native window controls.

🔧 Zero Dependencies

No React, no Tailwind, no webpack. Pure vanilla JavaScript, HTML Canvas 2D, and the browser's built-in Web Audio API. The entire app is the app.

🎧 Queue Management

Drag-to-reorder queue with track progression. Load up a playlist and let the visualizer ride through your entire collection.

How It Works

🔍 Pathfinding Engine

Type-agnostic shortest-path search across the actor→movie graph. Connect actor-to-actor, actor-to-movie, or movie-to-movie — the engine handles all combinations.

📊 Difficulty Levels

Curated levels from easy (household names) to expert (deep cuts). Each level has a target path length, actor pool, and difficulty rating. Plus custom level creation.

💡 Smart Suggestions

Context-aware hints that show which movies connect your current position to the goal, with popularity weighting and path-proximity scoring.

📸 Offline Snapshots

The frontend exports and caches the entire game graph to localStorage. Play without a backend connection — data syncs when available.

🎨 Polished UI

Purple gradient design with actor/movie cards, path visualization sidebar, suggestion buttons, and responsive layout. Built with React 19 and Vite.

🗃️ TMDB Enriched Data

Full metadata from The Movie Database API — posters, bios, ratings, release dates. Automatic backfill and enrichment pipeline.

System Architecture

⚡ Pipeline Context

Agents share state through a PipelineContext object that flows through the chain. Each agent reads from predecessors and writes enriched data for the next stage.

📡 SSE Streaming

Results stream to the browser in real time via Server-Sent Events. Watch each agent light up as it processes — no polling, no websockets.

🖥️ Dual Interface

Interactive browser dashboard with animated pipeline visualization, plus a full CLI for batch processing and scripting.

📈 Performance Tracking

Built-in LabTracker instrumentation — per-agent timing, memory usage, throughput metrics. The whole pipeline runs in ~50ms.

🔌 Agent Registry

Pluggable agent architecture with a central registry. Add new analysis agents without modifying the orchestrator.

🏠 100% Local

No cloud dependencies, no API keys, no LLMs. Everything runs on your machine. Pure Python rule-based analysis.

Capabilities

📦 Full Pipeline

One command runs the entire chain: ingest → decompile → scan → extract → scaffold → report. Or run each stage individually.

🔍 Endpoint Discovery

Scans decompiled source for HTTP endpoints, WebSocket connections, API keys, and authentication flows the game depends on.

🏗️ Mock Server Generation

Automatically generates a runnable Node.js server scaffold that can replace the original backend — keeping the game alive.

🎮 Game Preservation

Built for the long game. When servers shut down, mobile games die. This tool documents what was lost and builds the bridge to keep playing.

🔌 Plugin Architecture

Engine-specific logic lives in plugins. Add support for new game engines without touching the core pipeline.

📋 Rich Reports

Generates detailed Markdown and JSON reports documenting every endpoint, asset, and dependency discovered in the APK.

What Makes This Tick

🏛 Impossible Architecture

An isometric building with Penrose-style staircases. You go up the left, across the top, down the right — and arrive back at the bottom. The geometry is impossible. The CSS doesn’t care.

📜 Self-Reading Source

At runtime, JS reads this page’s own HTML, extracts real code fragments, and injects them as text flowing along the staircase paths. The building carries the blueprints that built it.

&#9889; Zero-Framework Stack

Vanilla JS, plain CSS, Vite for dev/build. No React, no Tailwind, no dependencies that’ll be deprecated before you finish reading this. Each page has its own visual theme.

🎨 Five Unique Themes

Cosmic purple for home, hot pink cinema for Co-Stars, CRT green terminal for Multi-Agent, amber archaeological dig for APK Archeologist, and this impossible cyan building for the page that carries itself.

🎮 In-Portfolio Demos

Every project runs live inside its portfolio page — no external links, no “visit the deployed site.” The Vibe Machine visualizer, Co-Stars graph, and more all play right here in the browser alongside their screenshots and write-ups.

🚀 Deploys Itself

Built with Vite, ships as static HTML. No server required. The thing describing itself is also the thing deploying itself. The ouroboros of web development.
